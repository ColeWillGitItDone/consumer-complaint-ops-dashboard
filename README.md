# Consumer Complaint Operations Dashboard

## Overview
This project analyzes publicly available CFPB consumer complaint data using Python in VS Code. The goal is to identify complaint trends, product concentration, issue concentration, and state-level patterns relevant to operations, service quality, and process improvement.

## Business Questions
- How has complaint volume changed over time?
- Which products generate the most complaints?
- Which issues appear most frequently?
- Which states show the highest complaint concentrations?
- Which product-issue combinations may represent process improvement opportunities?

## Outputs
- Cleaned complaint dataset
- KPI summary tables
- Monthly trend chart
- Top product chart
- Top issue chart

## Files
- `src/01.load_data.py` - loads and previews the raw data
- `src/02.clean_data.py` - cleans and standardizes the dataset
- `src/03.analyze_data.py` - creates KPI summary tables
- `src/04.export_outputs.py` - exports charts
